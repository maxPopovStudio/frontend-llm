# Інструкція з встановлення та запуску

## Вимоги

- Node.js версії 18 або вище
- npm або yarn
- Браузер з підтримкою WebGPU або WebAssembly (для transformers.js)

## Встановлення

1. Встановіть залежності:
```bash
npm install
```

або

```bash
yarn install
```

## Запуск

Для запуску в режимі розробки:

```bash
npm run dev
```

або

```bash
yarn dev
```

Застосунок буде доступний за адресою `http://localhost:3000`

## Збірка для продакшну

```bash
npm run build
```

або

```bash
yarn build
```

Зібраний застосунок буде в папці `dist/`

## Перегляд зібраного застосунку

```bash
npm run preview
```

## Примітки

- При першому запуску моделі LLM будуть завантажені з CDN (це може зайняти деякий час)
- Моделі кешуються в браузері для швидшого доступу при наступних запусках
- Для роботи з WebGPU потрібен сучасний браузер (Chrome 113+, Edge 113+, Safari 18+)
- Якщо WebGPU недоступний, система автоматично переключиться на CPU режим
